Hackabox

Goals:

To develop a MEAN stack website for my Hackabox project, including a forum built from scratch.

Challenges:

Building a forum from scratch with AngularJS (and other tech).

Learned:

Further developed my AngularJS and other MEAN tech skills. I had to go very in depth with AngularJS for the forum.

More:

This site was born out of an idea I had to create a gaming platform out of a Raspberry Pi (computer). While I wasn't really able to get going on the project, I did end up making a simple MEAN stack site. A MEAN stack consists of the technologies represented by the logos below. MongoDB (database), ExpressJS, AngularJS, and NodeJS. I made a forum from scratch, and a fully functional user system. It is responsive. You can click on the image, and it will bring you to a demo - feel free to create a user and use the forum, everything that is available on the demo is fully functional (including a MongoDB backend). Please proceed to the next slides for more details on this project.

This is a very simple native iOS app I developed. My idea was to create a game in which one would go somewhere and take a picture, and then someone would have to go find where you took the picture (with help from GPS) and take almost the same picture to complete the objective. The backend uses a console program called imageMagick to analyze the pictures to find out if they are the same. I used Parse as the meat of the backend to handle data that needs to be stored in a database, or retrieved from it. You can learn about Parse at parse.com
